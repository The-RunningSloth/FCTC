You == Software Engineer

Why web development? Because Full-Stack Software Engineers are in huge demand. The jobs are high-paying. Every major metropolitan area is hiring.

Source: Glassdoor, December 2022:

Software Engineers are also consistently in the top ten for happiest careers. You don't need a degree: you just have to commit the time.

From Cops To Coders uses the 100Devs online bootcamp. 100Devs is COMPLETELY FREE. You will learn all the fundamental skills you need to unlock a career as a software engineerincluding HTML, CSS, Javascript, React, Node, Express, MongoDB, and some computer science theory. You will build web pages, learn how to program, and create projects.

100Devs doesn't just teach you how to code -- it teaches you how to get a job. You'll create a portfolio web site, use social media, build your resume, and network your way into a new, happy, well-paid career.

All of the class videos and assignments are online, so you can work at your own pace. You can check out the first class here on youtube:
